# Tugas

Final adalah keyword yang digunakan untuk menyatakan bahwa suatu variabel hanya boleh diisi satu kali dan tidak dapat diubah lagi setelah diberi nilai tapi bisa di tentukan saat program berjalan.

Const adalah keyword yang digunakan untuk menyatakan bahwa suatu variabel memiliki nilai tetap yang sudah ditentukan sejak proses compile, atau nilai wajib di ketahui sebelum program berjalan.

1. Waktu Penentuan Nilai

final → nilai ditentukan saat runtime

const → nilai ditentukan saat compile time

2. Fleksibilitas Penggunaan

final

-Bisa diisi dari input

-Bisa dari hasil perhitungan

-Bisa dari fungsi

const

-Hanya bisa diisi nilai tetap

-Tidak bisa dari input atau fungsi runtime

3. Perubahan Nilai

final → tidak bisa diubah
const → tidak bisa diubah
Keduanya sama-sama immutable, tapi cara penentuannya berbeda.
